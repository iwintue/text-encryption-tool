## English Description

A Java application that implements various text-processing and cryptographic algorithms. It provides tools for secure text encryption, decryption, and basic cryptanalysis using statistical methods.

### Features
* **Multiple Ciphers Support:** Implementation of classic encryption algorithms (e.g., Caesar cipher, Vigenere cipher, or substitution ciphers).
* **Automated Decryption:** Features text-processing algorithms to restore original text.
* **Frequency Analysis:** Uses statistical frequency analysis techniques to break simple ciphers without knowing the key.

### Tech Stack
* **Language:** Java (JDK 17+)
* **Concepts:** Cryptography, String Processing, Data Structures (Maps/Lists for frequency counters).

## Описание на Русском <a name="russian-version"></a>
Приложение на языке Java, реализующее различные алгоритмы обработки текста и криптографии. Предоставляет инструменты для безопасного шифрования, расшифровки текста и базового криптоанализа с использованием статистических методов.
### Функционал
* **Поддержка нескольких шифров:** Реализация классических алгоритмов шифрования (например, шифр Цезаря, шифр Виженера или шифры подстановки).
* **Автоматическая расшифровка:** Алгоритмы обработки строк для восстановления исходного текста.
* **Частотный анализ:** Использование методов статистического частотного анализа для взлома простых шифров без необходимости знания ключа.
### Стек технологий
* **Язык:** Java (JDK 17+)
* **Концепции:** Криптография, обработка строк (String Processing), структуры данных (Maps/Lists для счетчиков частоты символов).
